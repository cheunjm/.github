## Adding New Tools

When a new tool is installed, add an entry above with:
- **Tool**: Name + link
- **Type**: GitHub App / GitHub Action / External Service
- **Purpose**: What it does
- **Repos**: Which repos it applies to
- **Status**: ✅ Active / 🔜 Pending / ❌ Removed
- **Installed**: Date
